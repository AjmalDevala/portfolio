## 🚀 Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/AjmalDevala/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm start
   ```

4. **View the Website**
   ```
   http://localhost:3000
   ```
